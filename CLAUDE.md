# Claude instructions for this repo

## Branch and PR workflow

- Development branch: `claude/conveyor-belt-ui-67oyg`
- After every commit (or batch of commits) to the branch, always open or update a pull request targeting `master`.
- Use the GitHub MCP tools (`mcp__github__create_pull_request`) to create the PR if one doesn't exist yet.
- If a PR already exists for the branch, do not create a duplicate — just push the commit.
